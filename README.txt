ng-seed
=======

Bare-bones angular &amp; bootstrap web app


TO LAUNCH FRONT END
1) in your terminal, type the command:
  $ python -m SimpleHTTPServer
2) then go to:
  http://0.0.0.0:8000/#/

TO LAUNCH SERVER
1)
	$ node server/bloom.js
2) then go to:
	http://localhost:3000/

IMPORTANT
- get bloomberg develop keys (bloomberg, *.crt, *.key)
- rename yours as bloomberg.crt, client.crt, client.key
- move into server/
